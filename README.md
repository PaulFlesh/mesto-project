# Проект: Место

## Ссылка на проект GitHub Pages: https://paulflesh.github.io/mesto-project/

### Навигация по файлам скриптов:
- modal.js - Взаимодействие с модальными окнами (открытие, закрытие),
- data.js - Начальный массив с карточками,
- cards.js - Взаимодействие формы создания нового места,
- validation.js - Валидация всех форм в проекте,
- profile.js - Редактирование профиля

### Изменения:
- Аватар и имя пользователя изменяются сразу после сабмита,
- Лайки обновляются в реальном времени,
- Карточка удаляется в реальном времени,
