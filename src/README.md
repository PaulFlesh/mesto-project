# Проект: Место

Проект в рамках четвертого месяца обучения Яндекс Практикум курса "Веб-разработчик плюс"
В проекте реализуются:
- Валидация форм;
- Модульность кода;
- Добавление функциональности на JavaScript;
- Настройка сборки Webpack;
- Взаимодействие с сервером.
